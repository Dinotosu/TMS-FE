2.  Залить верстку к себе в гит-репозиторий

P.S.:

- Генерируем access_token (https://docs.github.com/en/enterprise-server@3.4/authentication/keeping-your-account-and-data-secure/creating-a-personal-access-token)

- (если на уроке использовали свой компьютер то пропускаем) клонируем ваш github-репозиторий к себе на комп:
  заходим в ваш github -> заходим в ваш репозиторий -> нажимаем зеленую кнопку Code -> выбираем HTTPS -> копируем ссылку -> на своем компе заходим в папку где вы будите хранить всю работу в TMS (свой репозиторий) -> правая кнопка мыши -> git bash here -> пишем git clone ссылка_которую_скопировали (вместо пароля вводим access_token который сгенерировали) - ` git clone пропишите только один раз и все, каждый раз этого делать уже не нужно будет -> должен быть склонирован ваш удаленный репозиторий -> переходим в папку которую скопировали


- дальше для каждой домашки вы создаете в TMS-frontend новую папку (пример: для того чтобы сдать текущую домашку создаем в TMS-frontend папку lesson-5-homework -> в ней делаем верстку -> после того как закончили -> ВАЖНО:переходим в свой репозиторий, это папка TMS-frontend) и именно в ней нажимаем git bash here, т.к именно в этой папке проинициализирован git -> git status (просмотрели что изменили - все изменения красным) -> индексируем изменения (git add .) -> git status (все изменения должны проиндексироваться и стать зеленым) -> коммитим изменения (git commit -m "...")  -> пушим изменения в свой удаленный репозиторий (git push), вводим свой user-name, вместо пароля вводим сгенерированный access_token-> заходим на гитхаб и проверяем - все изменения должны появиться на github -> далее на гугл диск просто скидываем текстовый файл с ссылкой на свой репозиторий для того чтобы я понимал какие домашки можно смотреть,a какие еще какие не готовы)

- перед следующим занятием мне важно понимать что все разобрались и у всех получилось запушить в свой репозиторий чтоб мы не тратили на разбирательства время на занятиию

- в конце курса ваша папка(репозиторий) будет выглядеть примерно так:
TMS-frontend

    files of lesson-5 homework
  lesson-6
    files of lesson 6
  lesson-6-homework
    files of lesson 6 homework
  ...
